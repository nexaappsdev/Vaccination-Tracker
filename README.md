# Vaccination Tracker — Public Legal Pages

GitHub Pages site hosting the legal pages referenced from the
**Vaccination Tracker** Android app and Play Store listing.

## Live URLs

- **Privacy Policy** → https://nexaappsdev.github.io/Vaccination-Tracker/privacy-policy.html
- **Root (alias)** → https://nexaappsdev.github.io/Vaccination-Tracker/

## How it works

GitHub Pages serves `index.html` from this repo's `main` branch. The
same content is also available at `privacy-policy.html` for stable
Play Store / app references that match the canonical filename.

## Updating

Edit `privacy-policy.html` and `index.html` (keep them in sync), commit
to `main`, push. GitHub Pages re-publishes within a minute or two.
